# 微信小程序自动修复Agent - 可行性验证

## 功能说明

这是一个自动化错误捕获和修复的Agent原型，用于验证技术可行性。

### 核心功能

1. **编译错误捕获** - 通过微信CLI捕获编译错误
2. **运行时错误捕获** - 通过automator捕获运行时错误和截图
3. **AI自动分析** - 调用GPT-4o分析错误并生成修复方案
4. **自动应用修复** - 自动修改代码并重新验证

## 使用步骤

### 1. 安装依赖

```bash
cd scripts/auto-fix-agent
npm install
```

### 2. 配置环境变量

```bash
export OPENAI_API_KEY="your-api-key"
```

### 3. 确保微信开发者工具配置

- 打开微信开发者工具
- 设置 → 安全设置 → 开启"服务端口"
- 设置 → 通用设置 → 开启"允许命令行调用"

### 4. 运行测试

```bash
# 运行简单测试（自动创建错误并修复）
npm test

# 或直接运行Agent
node index.js
```

## 工作流程

```
1. 检查编译错误
   ↓ 有错误
2. AI分析 + 生成修复方案
   ↓
3. 自动应用修复
   ↓
4. 重新编译检查
   ↓ 编译通过
5. 启动小程序检查运行时错误
   ↓ 有错误
6. AI分析 + 截图
   ↓
7. 自动应用修复
   ↓
8. 重新运行检查
   ↓ 无错误
9. 完成 ✅
```

## 文件说明

- `config.js` - 配置文件（项目路径、AI配置等）
- `error-capture.js` - 错误捕获模块
- `ai-fixer.js` - AI修复模块
- `index.js` - 主入口
- `test-simple.js` - 简单测试脚本

## 限制说明

1. 需要微信开发者工具保持打开
2. 需要有效的OpenAI API Key
3. 复杂业务逻辑错误可能修复失败
4. 每次修复会调用AI API（有成本）
